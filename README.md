# Gemini Folders and Protected Files

Gemini Folders and Protected Files is a browser extension that improves Gemini chat organization, protected chat, and more basic sidebar usability.

## Features

- Create folders and nested subfolders to organize Gemini chats.
- Move chats between folders with drag-and-drop interactions.
- Protect pinned chats from accidental deletion by disabling delete actions for protected/pinned items.
- Preserve native Gemini chat menu behavior while layering protection logic where required.
- Collapse and expand folder trees with smoother visibility handling and hidden-item count indicators.
- Assign folder emojis using a searchable, category-based emoji picker sourced from local JSON data.
- Support browser-aware emoji/flag rendering behavior with compatibility fallback handling.
- Provide a cleaner sidebar presentation with improved spacing, alignment, and hover clarity.
- Include popup debug controls for visual diagnostics and logging when troubleshooting is needed.

## Load the Extension (Unpacked)

1. Open browsers' extensions page:
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select the project root folder (the folder that contains `manifest.json`).
